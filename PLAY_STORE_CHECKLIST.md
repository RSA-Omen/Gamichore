# GamiChore Play Store checklist

Use this checklist before submitting to Google Play.

## Policy setup

- Publish a public privacy policy URL.
- Complete the Data safety form in Play Console.
- Complete the Content rating questionnaire.
- Set ads declaration correctly (`No` unless ads are added).
- If targeting kids/families, complete Families policy requirements.

## Android release

- Keep target SDK aligned with current Play requirement.
- Increment `versionCode` for every upload.
- Build signed AAB for release.
- Test on at least one phone and one tablet profile.

## Security and privacy

- Parent controls are protected behind a parent PIN.
- Do not ship any secrets or private keys in app assets.
- Document all third-party network calls in privacy policy.
- Review backup behavior for child-related local data.

## Functional quality

- Verify no duplicate chore submissions can be approved twice.
- Verify shop image upload blocks oversized files.
- Verify offline launch and navigation works.
- Verify parent login and logout flow works every run.

## Store assets

- Replace placeholder icons with production 192x192 and 512x512 PNG icons.
- Prepare screenshots, feature graphic, and listing copy.
